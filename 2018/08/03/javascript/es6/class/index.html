class/extends实现

发表于 2018-08-03

分析下es6中class/extends的实现

babel转码es5后

总结：

es6继承: 先通过super关键字将父类实例的属性和方法添加到this上，再调用子类的构造函数完成对this的修饰。
具体实现上相比es5使用的构造函数复制，es6使用的是将子类构造函数的原型链指向父类的构造函数。
